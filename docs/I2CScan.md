# I²C Scan Plugin

The I²C Scan plugin searches for first `I2C_MAX_DEVICES` (see [plgI2CScan.cpp](/plgI2CScan.cpp)) 
devices in I²C bus. 

### Connection
|I²C bus|MultiSens Pin|Color|
|:---:|:---:|:---|
|GND|GND|Black|
|VIN|+5V|Red|
|SDA|P6|Yellow-Black|
|SCL|P7|Gray-Black|
